## Project Goal
Help people decide whether renting or buying an apartment in Geneva is financially optimal based on:
- rent prices,
- mortgage rates,
- predicted market appreciation (Machine Learning),
- personal financial situation.

## Key Features (planned)
- Download real estate data for Geneva (rent & sale prices).
- Predict future property prices (Regression model).
- Estimate break-even point (Monte Carlo simulation).
- Generate decision report: *"Renting becomes more expensive after X years."*

## Tech Stack
- Python
- pandas, numpy
- scikit-learn (ML models)
- matplotlib (visualizations)

## Deliverables
- ML model that predicts future prices
- PDF or dashboard showing results
